# To-Do List 

A simple and interactive **To-Do List** web application built with **HTML, CSS, and JavaScript**. This app allows users to add, edit, and delete tasks, with all tasks being stored in **local storage** so they persist across page reloads.

---

## 📝 Features

- Add new tasks with:
  - **Title**
  - **Date**
  - **Description**
- Edit existing tasks
- Delete tasks
- Tasks are saved in **local storage** so data is retained even after refreshing the page
- Clean and responsive design with a pop-up input form
- Overlay to focus on task input

---

## 💻 Technologies Used

- **HTML5** – For page structure
- **CSS3** – For styling and layout
- **JavaScript (Vanilla)** – For interactivity and local storage
